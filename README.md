### Instalasi
_Ikut langkah langkah ini :_
#### 1. Git Clone
`git clone https://github.com/fadhlurrahmanID/lucunah.git`
#### 2. Masuk Ke Folder Project
`cd namaFolder`
#### 3. Install React Script
`npm i react-scripts`
#### 4. Npm Start / Jalanin Hosting
`npm start`
#### 5. Open Browser
after npm start is done open `localhost:3000`
